== Description ==
A simple plugin for changing shipping labels in WooCommece cart and checkout. You have options for showing or hiding it completely and you can change the labels to the text that you want!

I can't guarantee that the labels you have changed match the legal obligations in your country, so please check this in advance. I am not responsible for the functionality of the plugin and the use is at your own risk and responsibility.


== Installation ==
Just download and activate it.
Settings can be found at WooCommerce > Change Shipping Labels.
